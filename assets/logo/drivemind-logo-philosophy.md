# DriveMind Logo Philosophy

## Movement name
Signal Gate

Signal Gate treats DriveMind as execution infrastructure rather than an app mascot.

The mark should communicate three actions:
- evidence enters
- a boundary gate evaluates it
- authorized execution moves forward

This matches the v0.7 product claim: DriveMind does not merely make agents calmer; it keeps the task inside the right boundary while forcing stronger execution when the model is settling too low.

## Visual rules

- Use strong architectural geometry.
- Prefer ink, slate, white, and one signal accent.
- Keep the mark readable at small sizes.
- Avoid glossy gradients, brain icons, robots, lightning, neural-network dots, and toy-like SaaS shapes.
- Keep image assets and SVG source assets aligned around the same Signal Gate concept.

## Palette

- Ink: `#071525`
- Signal teal: `#19D39A`
- Slate: `#607084`
- Surface: `#F7F9FC`
