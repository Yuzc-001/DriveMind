# DriveMind v0.1.0

DriveMind `v0.1.0` is the first public foundation release.

It turns DriveMind from a good idea into an installable, source-available product surface for calm, human-in-the-loop agent work.

## What ships in v0.1.0
- the first reusable DriveMind skill
- one-click install for Codex and Claude Code
- persistence, escalation, mode, review, and distillation references
- bilingual README entry points in English and Chinese
- governance docs for contributing, conduct, security, changelog, and trademark boundaries
- a minimal validation workflow for scripts and markdown links
- logo source assets and a prepared GitHub social preview asset
- source-available licensing under BUSL-1.1.

## Product direction
DriveMind is a calm reliability layer for AI agents.

It is designed to help agents:
- stay with meaningful work longer
- ask before crossing unclear boundaries
- keep humans informed during execution
- leave behind reusable lessons after the task.
