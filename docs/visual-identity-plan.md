# Visual Identity Plan

DriveMind v0.7 uses the **Signal Gate** identity.

## What it means

- signal enters
- boundary gate evaluates
- authorized execution moves forward

The identity should feel like agent infrastructure, not a toy app.

## Rules

- no brain icons
- no robot mascots
- no lightning symbols
- no generic AI network dots
- no glossy gradient app-icon style
- keep the mark inspectable and source-control friendly
